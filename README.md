# DropSynth

AI-powered dropshipping waitlist. Built with Vite + React + Supabase.

## Setup
1. Copy `.env.example` to `.env` and add your Supabase URL and anon key
2. `npm install && npm run dev`
3. Deploy to Vercel with VITE_SUPABASE_URL and VITE_SUPABASE_ANON_KEY env vars

See README.md in the project root for full launch guide.
